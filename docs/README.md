## What FCM<sub>PASS</sub> Does

<div class="grid cards" markdown>

-   :material-database:{ .lg .middle } **Catalogue**

    ---

    Store and manage your flow cytometers, light scatter reference materials, fluorescent beads, and rainbow particles in a centralized catalogue — reducing repetitive data entry and ensuring consistency.

    [:octicons-arrow-right-24: Learn more](cataloguing/README.md)

-   :material-chart-bell-curve-cumulative:{ .lg .middle } **Calibrate**

    ---

    Calibrate fluorescence and light scatter data into standardized units, enabling cross-platform comparisons and traceable MIFlowCyt and MIFlowCyt-EV reports.

    [:octicons-arrow-right-24: Learn more](calibration/README.md)

-   :material-tune-vertical:{ .lg .middle } **Optimize**

    ---

    Generate and analyze detector optimization data for both fluorescence and light scatter channels to get the best performance from your flow cytometer.

    [:octicons-arrow-right-24: Learn more](optimization/fluorescence/data-generation.md)

-   :material-file-chart:{ .lg .middle } **Report**

    ---

    Automatically generate transparent, shareable reports with detailed metadata — ready for publication and compliant with MIFlowCyt-EV guidelines.

    [:octicons-arrow-right-24: Learn more](calibration/process-fcs-files.md)

</div>
